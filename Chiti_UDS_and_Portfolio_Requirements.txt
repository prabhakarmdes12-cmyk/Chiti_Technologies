PART A: CHITI UNIFIED DESIGN SYSTEM (UDS V3)
Goal: A full-featured, enterprise-grade component architecture and governance model.
================================================================================

1. THE FOUNDATIONS (Core Variables & Rules)
   [ ] Design Tokens: Centralized JSON map linking hex codes to semantic layers.
   [ ] Semantic Color Protocol: The "Sapphire/Teal" DataViz safety protocol alongside semantic states (Success, Warning, Error).
   [ ] Typography Engine: Font-families (Outfit, Inter, JetBrains) mapped to specific DOM scales (H1-H6, Body, Label).
   [ ] The 8-Point Spatial Grid: Absolute geometrical spacing mapped to 8px intervals.
   [ ] Elevation & Depth: Z-index and shadow box mechanics defining physical layering.
   [ ] Motion & Haptics: Web Audio API integration and specific Cubic-Bezier physics timing arrays.

2. ASSEMBLED COMPONENTS (The Exhaustive React Kit)
   *A full design system must document all functional states of the following UI Nodes:*
   
   [ ] Form & Input Controls: 
       - Buttons (Primary, Outline, Ghost, Danger)
       - Text Inputs (Default, Focus, Invalid, Disabled)
       - Select/Dropdown Arrays
       - Hardware Toggles & Checkboxes
   
   [ ] Navigation Modules: 
       - Top Application Headers
       - Collapsible Sidebar Drawers
       - Breadcrumbs & Pagination Arrays
   
   [ ] Data Display & Layout: 
       - The 'ChitiCard' (3D Physics enabled)
       - Enterprise Data Tables (Sortable/Filterable)
       - Badges & Status Chips
       - Modals & Dialogue Overlays
       - Interactive Tooltips
       
   [ ] Feedback & State Handling: 
       - Toasts / Snackbars (Success/Failure states)
       - Progress Loaders & Spinners
       - Hydration Skeletons (ARIA compliant)

3. PATTERNS & GOVERNANCE
   [ ] Empty States logic (What happens when data is zero?).
   [ ] Validation & Error Boundaries.
   [ ] Figma Handoff Protocols: Auto-layout mandates vs free-dragging.
   [ ] Accessibility (A11y): Strict WCAG 2.1 compliance, Keyboard trapping logic, Tab-indexing.
